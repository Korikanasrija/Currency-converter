# 💱 Simple Currency Converter

A lightweight, beginner-friendly Currency Converter web application built using HTML5, CSS3, and modern Vanilla JavaScript (ES6+). This project utilizes a completely free, live API to fetch real-time currency exchange rates without requiring an API key.

## 🚀 Features

- **Live Exchange Rates:** Fetches up-to-date currency valuations instantly.
- **Key Currency Support:** Easily convert between USD, EUR, GBP, INR, JPY, CAD, and AUD.
- **Zero Configuration:** Uses a free, public API (`open.er-api.com`)—no API keys or sign-ups needed.
- **Responsive Design:** Clean, modern card interface that looks great on mobile, tablet, and desktop devices.
- **Error Handling:** Gracefully handles invalid inputs or network/API connectivity losses.

## 🛠️ Built With

- **HTML5:** Semantic markup for the app structure.
- **CSS3:** Custom styles, custom layout alignments, and smooth transitions.
- **JavaScript (ES6):** Asynchronous API data fetching using `async/await`, dynamic DOM manipulation, and event handling.
